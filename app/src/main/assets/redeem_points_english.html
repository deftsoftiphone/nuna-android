Each Like received by a post you create can be redeemed for ₹1 (1 Like = ₹1). Follow the steps given below to redeem the number of Likes received ("Likes") on your Nuna account:
 
👉 Create a PayTM account with the same phone number used to create your account on Nuna.
👉 Send a WhatsApp text message to +91 96671 35355 with the following message: REDEEM LIKES
👉 Nuna will calculate the number of valid "Likes", and transfer the associated cash amount in Indian Rupees (₹) to the PayTM account linked to the phone number, within 5 business days.
👉 "Likes" can be redeemed only once. Any "Likes" that have been redeemed once will be excluded from any future redemption requests.
👉 For each post, a minimum of 10 "Likes" are needed for those "Likes" to be redeemable for cash. For example, if a post has 9 "Likes", ₹0 can be redeemed for that post. On the other hand, if a post has 11 "Likes", ₹11 can be redeemed for that post.
